Welcome to my page

This application is about all things UFC. It connects with the official UFC API, which needs some real work actually, to provide up to date information on fighters, upcoming/past fights, and various UFC-related articles.


UFC has come a long way in regards to how I viewed it as a sport. It can definitely be a bit gruesome at times but there is a certain beauty in it that can be seen by those who are willing.

Developer Notes

  • The centerpiece of this application is AngularJS 1.x.
  • I use primarily Node.js and npm as the environment I develop in.
  • I will implement some testing into this application as well. I expect this application to be something I will work on from time to time.
  • Initially, there won't be much custom back-end involved for this application as I will rely entirely on UFC's API for data; however, I may implement user authentication through Rails' Devise gem if I want to take advantage of the UFC API's support for live events. I'd like to perhaps do something similar to what Yahoo sports does with their fantasy league, which I've never used before so may need to read up on.
